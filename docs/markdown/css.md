# SOMA Document Stylesheet

A beautiful, professional CSS stylesheet specifically designed for HTML documents generated by SOMA's markdown extension with `HtmlEmitter`.

## Features

- **Calm, professional design** - Clean typography and subtle colors
- **Optimized spacing** - Clear but not excessive (less than Google Docs)
- **Accessible typography** - Uses Atkinson Hyperlegible fonts
- **Optimized for SOMA HTML** - Styled for the exact HTML subset we generate
- **Responsive** - Works on mobile, tablet, and desktop
- **Print-friendly** - Optimized print styles included

## Typography

- **Body text**: Atkinson Hyperlegible Next (with fallbacks)
- **Code**: Atkinson Hyperlegible Mono (with fallbacks)
- **Base size**: 16px
- **Line height**: 1.6 (comfortable reading)

## Quick Start

### Option 1: Use the Template

1. Open `template.html` in a text editor
2. Generate HTML content with SOMA:
   ```soma
   (python) >use
   (markdown) >use
   >md.start
   md.htmlEmitter >md.emitter

   (My Document) >md.h1
   (This is content.) >md.p
   ```
3. Copy the generated HTML into the `<body>` section of `template.html`
4. Open in a browser

### Option 2: Link to CSS

Add this to your HTML `<head>`:

```html
<link rel="stylesheet" href="soma-document.css">
```

### Option 3: Inline the CSS

For self-contained documents, copy the CSS into a `<style>` tag in your HTML.

## Font Installation

The stylesheet uses Atkinson Hyperlegible fonts. To get the best experience:

1. **Download fonts**:
   - Atkinson Hyperlegible Next (variable-width)
   - Atkinson Hyperlegible Mono (fixed-width)

2. **Install locally** or **host on your server**

3. **Fallbacks included**: The stylesheet includes web-safe fallbacks if custom fonts aren't available

## Supported Elements

The stylesheet is optimized for the HTML elements generated by SOMA's `HtmlEmitter`:

### Text Formatting
- `<h1>`, `<h2>`, `<h3>`, `<h4>` - Headings
- `<p>` - Paragraphs
- `<strong>` - Bold text
- `<em>` - Italic text
- `<code>` - Inline code
- `<a>` - Links

### Lists
- `<ul>`, `<ol>`, `<li>` - Unordered and ordered lists
- Nested lists supported

### Code Blocks
- `<pre><code>` - Formatted code blocks
- Syntax highlighting friendly

### Tables
- `<table>`, `<thead>`, `<tbody>`, `<tr>`, `<th>`, `<td>`
- Alignment support (left, center, right)
- Zebra striping for readability

### Other
- `<blockquote>` - Quoted text
- `<hr>` - Horizontal rules

## Customization

The stylesheet uses CSS custom properties (variables) for easy customization:

```css
:root {
  --font-body: "Atkinson Hyperlegible Next", sans-serif;
  --font-mono: "Atkinson Hyperlegible Mono", monospace;
  --color-text: #2c3e50;
  --color-link: #2563eb;
  --space-md: 1.2rem;
  /* ... and more */
}
```

Edit these values to customize colors, fonts, and spacing.

## Design Philosophy

### Spacing
- **Paragraphs**: 1.2rem bottom margin (comfortable but not excessive)
- **Headings**: Larger top margins for section separation
- **Lists**: Balanced spacing for readability

### Colors
- **Text**: #2c3e50 (dark but not harsh black)
- **Headings**: #1a252f (slightly darker for hierarchy)
- **Links**: #2563eb (professional blue)
- **Code backgrounds**: Subtle grays for distinction

### Typography Choices
- **Atkinson Hyperlegible** - Designed specifically for readability
- **Optimal line height** - 1.6 for body text, 1.3 for headings
- **Appropriate font sizes** - Hierarchical and comfortable
